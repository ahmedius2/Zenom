Grafik Penceresi

Grafik penceresi kayıt değişkenlerinin değişimlerini zamana bağlı olarak iki boyutlu şekilde gösterir.  Değişimler benzetim koşumu esnasında anlık olarak veya durdurulduktan sonra gösterilebilir. Grafik üzerinde yakınlaştırma, uzaklaştırma ve dolaşım imkanı vardır. Ayrıca grafik Zenom Plot File(.zplot), resim(.jpeg, .png) ve Matlab (.m) dosya formatlarına uygun olarak kayıt edilebilir ve Zenom Plot File(.zplot) formatındaki dosyalar grafiğe yüklenebilir.

Zenom ana penceresindeki View menüsünden Plot seçimi ile aşağıdaki resimde gösterilen grafik penceresi açılır.

Grafiğe Kayıt Değişkeni Ekleme/Silme

Grafik alana kayıt değişkeni eklemek için,

  1. Plot menüsünden Attach Log Variable… seçilir veya araç çubuğundaki simgesine tıklanır. Aşağıda gösterilen Select a Variable(s) to Attach penceresi açılır.

  2. Listeden istenilen kayıt değişkenleri seçilir. Ctrl tuşuna basılarak birden fazla seçim yapılabilir.
  3. OK butonuna basılır. Seçilen kayıt değişkenleri grafikte gösterilir ve sağ tarafta yer alan listeye eklenir.

Grafik penceresinin sağ tarafında yer alan kayıt değişkenleri listesindeki düğmesine basılarak ilgili kayıt değişkenine ait çizim kaldırılabilir.

Kayıt değişkenleri listesinde çizim rengini belirten renk düğmesine basılarak çizim rengi değiştirilebilir.

Ölçeklendirme ve Navigasyon

Grafik alanın içerisinde farenin sol tuşuna basılıp bu tuş bırakılmadan fare hareket ettirilerek grafik üzerinde dolaşılabilir. Ayrıca farenin sağ tuşuna basılıp bu tuş bırakılmadan fare hareket ettirilerek veya fare tekerleği kullanılarak grafik alanda yakınlaştırılma/uzaklaştırılma yapılabilir.

AutoScale modu aktif iken x-ekseninde belirtilen zaman aralığındaki kayıt değişkeni değerleri gösterilir. Örneğin zaman aralığı 20 saniye girildiğinde grafik ekranda son 20 saniyelik veri sergilenir. AutoScale modu Plot menüsünden veya araç çubuğundaki düğmesi ile aktif hale getirilir.

Zaman aralığını değiştirmek için,

  1. Plot menüsünden Set Time Interval seçeneği seçilir veya araç çubuğundaki simgesine tıklanır. Aşağıda gösterilen Set Time Interval penceresi açılır. 

  2. İstenilen zaman değeri girilir.
  3. OK düğmesine basılır. X-eksenindeki zaman aralığı girilen değere göre değişir.

Grafiği Kaydetme

Grafikteki çizimler Zenom Plot(.zplot) dosya formatı ile kayıt edilip, tekrar geri yüklenebilir.
Grafiği kaydetmek için,

  1. File menüsünden Save seçeneği seçilir. Save Zenom Plot File penceresi görünür.
  2. İstenilen dosya ismi girilir.
  3. OK düğmesine basılır.

Grafiği açmak için,

  1. File menüsünden Open seçeneği seçilir. Open Zenom Plot File penceresi görünür.
  2. İstenilen dosya seçilir.
  3. OK düğmesine basılır.

Grafiği Farklı Formatlarda Kaydetme

Grafik, resim (.jpeg, .png) veya Matlab (.m) dosya formatı olarak kaydedilebilir. Grafiği farklı formatta kaydetmek için,

  1. File menüsünden Export seçeneği seçilir. Export Zenom Plot penceresi görünür.
  2. İstenilen dosya ismi girilir.
  3. İstenilen dosya formatı seçilir.
  4. OK düğmesine basılır.
